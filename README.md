# Angular 4 - Timer
I built the Timer(Start/Pause, Reset, Wait) using Angular4 and Observable.

- install dependencies
```
npm install
```

- fire up dev server
```
npm start
```
open browser to [`http://localhost:8080`](http://localhost:8080)

### Running Unit Tests
```
npm test
```
### Running E2E Tests
```
npm run webdriver
npm run e2e
```


[angular4]: https://angular.io/
